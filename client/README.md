# Nguzza Client

Frontend application for Nguzza marketplace, built with React + Vite.

## Features

- Responsive home page with improved cross-device footer.
- Flash sales section powered by admin-configured products.
- Category discovery, product cards, wishlist, cart, and checkout flows.
- Admin dashboard with flash sale controls.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```
