# Admin Dashboard Guide

The admin dashboard includes these tabs:

- **Monitor**: high-level platform analytics.
- **Users**: manage user roles and bans.
- **Products**: approve, delete, and quickly reset flash state.
- **Orders**: update order statuses and cancellations.
- **Inventory**: maintain crop types and livestock breeds.
- **Flash Sales**: create, schedule, and reset promotional flash campaigns.
- **Vendors**: approve or reject vendor registrations.

## Flash Sales Tab

The Flash Sales tab is the recommended way to manage short-term promotions:

- Select approved products.
- Choose campaign expiry time.
- Reset existing campaigns automatically if needed.
- Monitor currently active campaigns.
- Reset one product or all products in one click.
