## 🆘 Troubleshooting

### Can't run app?

```bash
flutter clean
flutter pub get
flutter pub run build_runner build
flutter run
```

### API not connecting?

- Ensure backend running: `npm run dev` (from api folder)
- Check base URL in `lib/services/api/api_client.dart`

### Build errors?

- Run: `flutter pub cache clean && flutter pub get`
- Then: `flutter pub run build_runner build`
